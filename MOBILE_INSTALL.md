# Android / Mobile Version

Это PWA-версия. Ее можно открыть на телефоне как сайт и добавить на главный экран Android.

## Локальный запуск

```bash
npm install
npm run dev
```

## Публикация в интернете через Vercel

1. Создать аккаунт на https://vercel.com
2. Загрузить проект в GitHub
3. В Vercel нажать `Add New Project`
4. Выбрать репозиторий
5. Нажать `Deploy`

После этого приложение будет доступно по ссылке вида:

```text
https://portfolio-tracker.vercel.app
```

## Установка на Android

1. Открыть сайт в Chrome на телефоне.
2. Нажать три точки справа сверху.
3. Выбрать `Добавить на главный экран` или `Install app`.
4. Иконка появится как обычное приложение.
