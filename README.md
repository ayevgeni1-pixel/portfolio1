# Portfolio Tracker MVP

Локальное веб-приложение для отслеживания портфеля.

## Запуск

```bash
npm install
npm run dev
```

После запуска открыть адрес, который покажет терминал, обычно:

```bash
http://localhost:5173
```

## Что уже есть

- Дашборд портфеля
- Таблица позиций
- Расчет стоимости
- Расчет дневного изменения
- Расчет прибыли/убытка
- График стоимости
- Распределение портфеля
- Блок Daily AI Brief
- Поддержка USD и ILS

## Что дальше

1. Вынести портфель в отдельный файл `portfolio.json`.
2. Подключить реальные котировки через API.
3. Добавить сохранение истории.
4. Добавить ежедневный отчет в Telegram/email.
5. Добавить анализ риска перед пенсией.
